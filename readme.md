### 🎯 二、课程简介

本课程面向人工智能时代的软件工程新范式，结合 The Modern Software Developer 的现代软件开发理念，以 Trae（AI 原生 IDE）为核心实践平台，系统讲授现代软件开发流程、AI 辅助编程方法与人机协作开发模式。

通过 **「理论 + 实验 + 项目」** 的混合教学模式，帮助你从「传统编码者」转型为「AI 驱动的软件系统构建者」，培养你的软件工程素养、智能开发能力与工程创新能力！

---

### 🎓 三、课程培养目标

✅ **理解现代软件开发**：掌握工程流程与技术体系
✅ **AI 辅助编程能力**：熟练使用 Trae 原生开发环境
✅ **自然语言驱动开发**：用语言高效构建软件系统
✅ **开发方式对比分析**：评估不同开发方式的效率与质量
✅ **人机协作开发**：具备「人 + AI」协作的工程实践能力
